# 線性回歸（Linear Regression）與 Keras 3 範例專案

本專案是一個教學導向的線性回歸範例，旨在展示三種線性回歸的實現方法：
1. **手動梯度下降實現 (From Scratch)**：使用 NumPy 從零開始編寫多元梯度下降，說明權重與偏差的更新數學邏輯。
2. **Scikit-Learn 官方實現**：使用業界標準機器學習套件進行快速建模與評估（基於最小平方法 OLS）。
3. **Keras 3 實現（JAX 後端）**：使用全新的 Keras 3 多後端深度學習框架，建立極簡單神經元網路，並以 JAX 作為後端引擎進行訓練。

三者會在隨機生成的線性噪音數據集上進行擬合，並比較預估權重、偏差、均方誤差 (MSE) 以及 $R^2$ 決定係數。最後，程式會將擬合曲線與手動/Keras 的損失收斂軌跡繪製成圖表儲存。

## Keras 3 的核心程式碼邏輯

在 `src/main.py` 中，Keras 3 的模型宣告與訓練非常簡潔：

```python
import os
# 設定 Keras 後端為 JAX (在導入 keras 之前設定)
os.environ["KERAS_BACKEND"] = "jax"
import keras

# 1. 建立一個最簡單的單層單神經元線性神經網路
keras_model = keras.Sequential([
    keras.layers.Input(shape=(1,)),
    keras.layers.Dense(1)  # output = w * x + b
])

# 2. 編譯模型，指定優化器 (SGD) 與損失函數 (MSE)
keras_model.compile(optimizer=keras.optimizers.SGD(learning_rate=0.01), loss="mse")

# 3. 訓練模型 (以靜音模式 fit)
keras_model.fit(X, y.reshape(-1, 1), epochs=500, batch_size=16, verbose=0)

# 4. 提取訓練完成後的 w 與 b 參數
keras_weights, keras_biases = keras_model.layers[0].get_weights()
w_keras = keras_weights[0][0]
b_keras = keras_biases[0]
```
